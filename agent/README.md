# Week 6 — ミニサイト

このフォルダは 100days_challenge の第6週用に作成した簡単な静的サイトです。

ファイル:

- `index.html` — サイト本体
- `styles.css` — スタイル
- `script.js` — テーマ切替スクリプト（ダーク/ライト）

使い方:

1. `week6` フォルダに移動します。

   ```powershell
   cd .\week6
   ```

2. 既定のブラウザで `index.html` を開きます（PowerShell のコマンド例）:

   ```powershell
   start .\index.html
   ```

ローカルでの練習や配布のために自由に編集してください。
